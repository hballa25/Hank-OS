# Starter Prompt 1 — Research before you build

> The author used the `/last30days` skill by Matt Van Horn for the research sweep (scours Reddit, X, YouTube, Hacker News for the last 30 days of real usage). This prompt asks for the same thing in plain words.

```
research how people are building second brain / memory systems for claude code right now - reddit, x,
youtube, hacker news, the lot. whats actually working, whats hype. then scan my whole workspace and
describe how my files, folders and memory are organised today. combine the two into a short plan for
a second brain that fits THIS workspace. before you build anything, ask me at least 3 questions about
how i actually work.
```
